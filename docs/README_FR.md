---

## Introduction

- Basée sur le runtime C# de Godot, cette traduction de la documentation d’aide permet de consulter en temps réel, dans la langue locale, l’utilisation des fonctions ainsi que leurs appels.

## Instructions d’utilisation

- Trouvez la langue et la version correspondantes.

- Par exemple :

1. Recherchez dans le répertoire `Chinese` du répertoire Git actuel.

2. Trouvez la version correspondante. Par exemple, s’il s’agit de `v4.6.1 v2`

3. Trouvez `GodotSharp.?.xml` et `GodotSharpEditor.?.xml`

4. Copiez-les vers :
    - Sous Windows :
        1. `C:\\Users\\votre_nom_utilisateur\\.nuget\\packages\\godotsharp\\4.6.1\\lib\\net8.0\\`
        2. `C:\\Users\\votre_nom_utilisateur\\.nuget\\packages\\godotsharpeditor\\4.6.1\\lib\\net8.0\\`
    - Sous Mac ou Linux :
        1. `/Users/votre_nom_utilisateur/.nuget/packages/godotsharp/4.6.1/lib/net8.0/`
        2. `/Users/votre_nom_utilisateur/.nuget/packages/godotsharpeditor/4.6.1/lib/net8.0/`

5. Sauvegardez `GodotSharp.xml` et `GodotSharpEditor.xml`, puis supprimez-les.

6. Renommez `GodotSharp.?.xml` en `GodotSharp.xml` et `GodotSharpEditor.?.xml` en `GodotSharpEditor.xml`, puis copiez-les dans ce répertoire.

7. C’est terminé. Ouvrez ensuite votre projet pour voir la langue correspondante.
