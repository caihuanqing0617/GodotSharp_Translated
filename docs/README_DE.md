## Einführung

- Basierend auf der Godot-C#-Laufzeitumgebung ermöglicht diese Übersetzung der Hilfedokumentation, die Verwendung und Aufrufe von Funktionen in Echtzeit in der lokalen Sprache anzuzeigen.

## Anleitung

- Suchen Sie die entsprechende Sprache und Version.

- Zum Beispiel:

1. Suchen Sie im Verzeichnis `Chinese` im aktuellen Git-Verzeichnis.

2. Suchen Sie die entsprechende Version. Wenn es `v4.6.1 v2` ist

3. Suchen Sie `GodotSharp.?.xml` und `GodotSharpEditor.?.xml`.

4. Kopieren Sie sie nach:
    - Unter Windows:
        1. `C:\\Users\\IhrBenutzername\\.nuget\\packages\\godotsharp\\4.6.1\\lib\\net8.0\\`
        2. `C:\\Users\\IhrBenutzername\\.nuget\\packages\\godotsharpeditor\\4.6.1\\lib\\net8.0\\`
    - Unter Mac oder Linux:
        1. `~/.nuget/packages/godotsharp/4.6.1/lib/net8.0/`
        2. `~/.nuget/packages/godotsharpeditor/4.6.1/lib/net8.0/`

5. Sichern Sie Ihre `GodotSharp.xml` und `GodotSharpEditor.xml` und löschen Sie sie anschließend.

6. Benennen Sie `GodotSharp.?.xml` in `GodotSharp.xml` und `GodotSharpEditor.?.xml` in `GodotSharpEditor.xml` um und kopieren Sie sie in dieses Verzeichnis.

7. Fertig. Öffnen Sie dann Ihr Projekt, und Sie können die entsprechende Sprache sehen.
