## Introducción

- Basada en el runtime C# de Godot, esta traducción de la documentación de ayuda permite consultar en tiempo real, en el idioma local, el uso de las funciones y cómo invocarlas.

## Instrucciones de uso

- Encuentra el idioma y la versión correspondientes.

- Por ejemplo:

1. Busca en el directorio `Chinese` del directorio Git actual.

2. Encuentra la versión correspondiente. Si es `v4.6.1 v2`

3. Encuentra `GodotSharp.?.xml` y `GodotSharpEditor.?.xml`.

4. Cópialos a:
    - En Windows:
        1. `C:\\Users\\tu_nombre_de_usuario\\.nuget\\packages\\godotsharp\\4.6.1\\lib\\net8.0\\`
        2. `C:\\Users\\tu_nombre_de_usuario\\.nuget\\packages\\godotsharpeditor\\4.6.1\\lib\\net8.0\\`
    - En Mac o Linux:
        1. `~/.nuget/packages/godotsharp/4.6.1/lib/net8.0/`
        2. `~/.nuget/packages/godotsharpeditor/4.6.1/lib/net8.0/`

5. Haz una copia de seguridad de `GodotSharp.xml` y `GodotSharpEditor.xml` y elimínalos.

6. Renombra `GodotSharp.?.xml` como `GodotSharp.xml` y `GodotSharpEditor.?.xml` como `GodotSharpEditor.xml`, y cópialos a ese დირექტorio.

7. Listo. Luego abre tu proyecto y podrás ver el idioma correspondiente.
