# Godot CSharp RunTime Help Document To Other Language

Godot C# 运行库的帮助文档的大语言模型的翻译到其它国家的语言

## 介绍

-   基于Godot C# 运行库的帮助文档的翻译,可以实时的查看函数本地语言的用法及调用.
    
## 使用说明

-   找到对应的语言和版本
    
-   譬如说:
    
  1.  在当前的Git目录 Chinese 目录下面找
    
  2.  找到对应的版本 如果是 v4.6.1
    
  3.  找到 GodotSharp.?.xml
    
  4.  复制到:
      - 在 Windows 系统下:   C:\\Users\\你的用户名\\.nuget\\packages\\godotsharp\\4.6.1\\lib\\net8.0
      - 在 Mac 系统下:       /Users/你的用户名/.nuget/packages/godotsharp/4.6.1/lib/net8.0
      - 在 Linux 系统下:     /Users/你的用户名/.nuget/packages/godotsharp/4.6.1/lib/net8.0
          
  6.  备份你的 GodotSharp.xml 且删除
    
  7.  把找到的 GodotSharp.?.xml 重命名为 GodotSharp.xml 且复制到该目录
    
  8.  完成 然后打开你的项目就可看到你所对应的语言了
